Подробное описание

Родительский класс для всех классов пакета.

Содержит основные функции, используемые тестами: Авторизация, установка ожиданий, инициализация драйвера, выход из аккаунта, переход "домой" и проверки видимости элемента

Переключение на авторизацию по логину\паролю
