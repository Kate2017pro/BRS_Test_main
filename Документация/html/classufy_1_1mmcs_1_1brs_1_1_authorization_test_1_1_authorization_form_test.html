void ufy.mmcs.brs.AuthorizationTest.AuthorizationFormTest.getDriver ( @Optional("chrome") String  browser)

Инициализация

Runs this method before the first test method in the current class is invoked. Инициализация драйвера и установка ожиданий. По-умолчанию используется браузер хром.

Аргументы
browserПередается из xml-файла для выбора браузера, в котором запустятся тесты. По-умолчанию = chrom
См. также
tearDown, Helper::timeouts_set, Helper::get_chrome_driver, Helper::get_firefox_driver
